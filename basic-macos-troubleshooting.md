# Basic macOS Troubleshooting

## Common Issues

- Slow performance
- Application crashes
- Wi-Fi problems
- Peripheral devices not detected
- Login issues

## Troubleshooting Steps

### 1. Gather Information

- What is the user experiencing?
- When did the issue start?
- Has anything changed recently?

### 2. Restart the Mac

Many temporary issues are resolved with a restart.

### 3. Check Available Storage

- Open System Settings
- Review available disk space
- Free space if storage is critically low

### 4. Verify Software Updates

- Open System Settings > General > Software Update
- Install available updates if appropriate

### 5. Check Network Connectivity

- Confirm Wi-Fi connection
- Test internet access using Safari
- Forget and reconnect to the wireless network if necessary

### 6. Test the Application

If one application is affected:

- Force quit and reopen
- Check for updates
- Test with another user account if available

### 7. Check External Devices

- Disconnect and reconnect peripherals
- Test alternative ports or cables
- Confirm devices appear in System Information

### 8. Escalate If Required

Escalate if:

- Hardware failure is suspected
- Operating system corruption is suspected
- The issue cannot be reproduced or resolved through standard troubleshooting

## Resolution Confirmation

- Confirm the issue is resolved with the user
- Document actions taken
- Update and close the ticket
