# Troubleshooting Slow Computer Performance

## Problem
User reports that their computer is running slowly.

---

## Possible Causes

- Too many startup programs
- High CPU or memory usage
- Low disk space
- Background applications running

---

## Troubleshooting Steps

1. Restart the computer  
   - Clears temporary processes and memory usage  

2. Check Task Manager (Windows) / Activity Monitor (Mac)  
   - Identify high CPU or memory usage  
   - Close unnecessary applications  

3. Disable unnecessary startup programs  
   - Prevents slow boot times  

4. Check available disk space  
   - Ensure at least 15–20% free space  

5. Run antivirus scan  
   - Rule out malware or unwanted software  

---

## Resolution

Performance improves after closing unnecessary applications and disabling startup programs.

---

## Notes

If issue persists, escalate to further diagnostics (hardware checks, OS reinstall, etc.)
